Phase 3

Approfondir le CSS

Dans cette phase, nous allons continuer à mettre en pratique le CSS pour permettre un aspect professionnel et compléter la navigation du site.

Site web, plusieurs pages

Dans la phase 3, nous allons dépasser une unique page web et mettre en place un site multi-pages. Vous devrez donc ajouter des liens entre les pages et s'assurer que l'ensemble reste fonctionnel.

Recherche sur votre entreprise

Vous allez devoir effectuer de la recherche d'information sur votre future entreprise. À titre d'information voici les entreprises qui emploient le plus de nos stagiaires:

  • Skytech
  • Globalia
  • Mediagrif

1 point Création de la page "Mon entreprise" et p3.css

Structure de votre projet

En partant d'une copie de votre page d'accueil, vous devez commencer une nouvelle page.

Vous devez également créer un nouveau fichier css p3.css pour y inclure les règles de cette phase. Il ne devra y avoir aucune copie de règle issue de p2.css dans p3.css. Vous devez inclure les 2 fichiers css dans vos pages html.

La structure de votre répertoire doit maintenant ressembler à :

  • dupont/
    • index.html
    • entreprise.html
    • css/
      • p2.css
      • p3.css
    • images/
      • portrait.png
Pointage
  • 1 point création de la page entreprise, du fichier p3.css

10 points Contenu la page "Mon entreprise"

Entreprise

Adapter le contenu de la page d'index pour y intégrer dans vos propres mots, un texte décrivant les éléments suivants pour une compagnie réelle:

  • Nom de l'entreprise
  • Année de création
  • Activité principale
  • Nombre d'employés
  • Carte Google Maps pointant la localisation de l'entreprise
Métier

Vous devez ensuite décrire votre métier en informatique dans votre entreprise en 2 ou 3 phrases en utilisant vos mots.

Trouvez au minimum 2 offres d'emploi pour le même métier (potentiellement dans des compagnies différentes) et ajoutez les dans une liste html non ordonnée avec les liens.

Indiquez dans une liste HTML 3 compétences communes aux 2 offres d'emploi que vous avez retenues.

Regardez la copie d'écran final pour l'apparence attendue.
Pointage Entreprise:
  • 1 point qualité du français
  • 4 points éléments de contenu réels. Vous devez inclure des liens vers les pages ayant servi de source.
  • 1 point carte Google Maps
Métier:
  • 1 point qualité du français
  • 1,5 point Offres d'emploi réelles et pertinentes. Vous devez inclure des liens vers les pages ayant servi de source.
  • 1,5 point Compétences communes bien identifiées

4 points Navigation

Navigation

Vous devez intégrer une zone de navigation entre pages. Cette div doit se situer entre top et middle.

Le menu doit contenir 4 éléments qui pointent chacun vers une page:

  • Accueil : pointe vers index.html
  • Mon entreprise : pointe vers entreprise.html
  • Ma formation : pointe vers formation.html (à venir)
  • Me contacter : pointe vers contact.html (à venir)

Chaque élément doit avoir une bordure avec des coins arrondis et de l'espace. Référez-vous à l'image pour l'aspect à reproduire. Vous devez choisir vos propres couleurs.

Vous devez inclure le menu de navigation dans les 2 pages existantes.

Regardez la copie d'écran final pour l'apparence attendue.
Pointage
  • 1 point menu dans les 2 pages
  • 1 point contient 4 liens dont 2 fonctionnels
  • 1 point la navigation est centrée dans la page
  • 1 point le style du menu est respecté
