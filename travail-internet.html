Projet : but global et phase 1

Votre site web dans 10 ans

Le but de ce site est de vous décrire dans 10 ans. Vous allez produire plusieurs pages web qui viseront à décrire

  • votre formation (collégial, universitaire, certifications etc.)
  • votre situation professionnel à travers l'entreprise pour laquelle vous travaillez
  • votre métier et votre quotidien.

Le travail permettra également d'illustrer vos compétences en HTML et en CSS.

Phases du projet
  • Phase 1 : page d'accueil, structure HTML, id et class
  • Phase 2 : CSS, style et mise en page
  • Phase 3 : multi-pages, navigation et métier
  • Phase 4 : formulaire, tableau, formation et BootStrap
Phase 1

Pendant la phase 1, vous allez mettre en place la structure de base de la page d'accueil et compléter quelques paragraphes de description général.

Vous devrez également apprendre à utiliser le validateur HTML5 afin de produire une page valide.

Finalement vous devrez comprendre la notion d'indentation du code.

1 point Structure du projet

Structure de votre projet
Vous devez créer un répertoire avec votre nom de famille qui devra lui-même contenir 2 sous répertoires. Un pour les fichiers CSS que vous produirez et un autre pour les images qui illustreront votre site. Vous devez ensuite créer le fichier texte index.html pour votre page d'accueil. La structure finale devra ressembler à
(pour un étudiant appelé Dupont)
  • dupont/
    • index.html
    • css/
    • images/

3 points Structure de la page index.html : head

Structure de la balise head
Vous devez utliser l'encodage UTF-8 pour l'ensemble de votre page.
Vous devez donner le title suivante à votre page "Professions informatiques : accueil"
Pointage
  • 1 point structure du head
  • 1 point titre
  • 1 point déclaration encodage

7 points Structure de la page index.html : body

Structure de la balise body
Vous devez reproduire la section suivante:
  • body
    • div avec id top
    • div avec id middle
      • div avec id middle-center
        • div avec id menu
          • Un titre h1 "Liens"
          • Un lien vers votre profil Facebook
          • Un lien vers la page du département d'info
        • div avec id content
          • un titre de niveau h1 : "Moi prénom nom"
          • un paragraphe vide (voir plus loin)
          • un titre de niveau h1 : "Mon projet"
          • un paragraphe vide (voir plus loin)
    • div avec id bottom
Pointage
  • 1 point pour le top
  • pour le middle
    • 2 points pour le menu
    • 3 points pour le content
  • 1 point pour le bottom

5 points Contenu de la section Moi

Vous devez vous présenter en imaginant votre situation dans 10 ans.

Vous devez produire un texte contenant entre 80 et 120 mots.

Vous devez indiquer vos 3 forces en terme de compétence dans une liste HTML.

Vous devez indiquer vos 3 points à améliorer en terme de compétence dans une liste HTML.

Pointage
  • 1 point pour le français
  • 1 point pour le respect de la longueur
  • 1,5 points pour vos forces
  • 1,5 points pour vos points à améliorer

2 points Contenu de la section "Mon projet"

Vous devez décrire le projet en mentionnant:
  • l'intitulé du cours ainsi que son numéro
  • l'objectif d'apprentissage des différents métiers et formations en informatique
  • L'objectif d'apprentissage de HTML et CSS
Votre texte doit faire entre 50 et 80 mots.
Pointage

  • 1 point pour le respect de la longueur
