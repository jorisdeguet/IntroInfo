Bootstrap

Avant la séance (2h)

Bootstrap est une feuille de style permettant de faciliter la mise en page d'une page web d'apparence professionnelle rapidement. Vous devez:

  • Explorer le site web de Bootstrap ici (max 5 minutes)
  • Regarder et reproduire les video sur Bootstrap ci-dessous
Pendant la séance
Vous compléterez les exercices non couverts pendant votre travail d'avant séance.
Vous devrez également commencer les pages Bootstrap du TP.
Exercices de la semaine

Exercice A

Réalisez les exercices des vidéos et assurez d'obtenir un résultat adaptatif ("responsive") en redimensionnant la page

Exercice B

Créez une page grille.html qui reproduit l'image ci-dessous en utilisant bootstrap.

Exercice C

Créez une page hockey.html qui reproduit l'image ci-dessous en utilisant bootstrap. Vous devez trouver les images sur Internet et vous choisirez la couleur de fond de votre choix.
